# EXP13 Langevin Falsification Modes

Question: do real SGD and Gaussian Langevin surrogates produce the same
variance dynamics in flat and sharp Hessian eigenspaces?

This is intended as the future main falsification experiment. The current MLP
version is a fast development target; a later paper-ready variant should move
the same protocol to NanoGPT or a larger CNN/ResNet setting.

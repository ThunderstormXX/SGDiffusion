# EXP16 Langevin Evidence Toy

Controlled stochastic-quadratic experiment comparing trajectory log-evidence
assigned by two Gaussian surrogate transition models:

- standard Langevin: covariance `eta^2 Sigma(w)`;
- modified Langevin: covariance `eta^2 [Sigma(w) + G(w)G(w)^T]`.

The trajectory itself is generated by the exact discrete SGD update on the
stochastic quadratic loss. The experiment also verifies that the closed-form
Gaussian log-ratio equals the direct density-ratio computation.
